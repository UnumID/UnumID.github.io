<Admonition type="warning" title="Sandbox Birth Date Value">
    **In [Sandbox](./environments#sandbox), for `birthDate` you need to use a value of `"1989-08-01"` to pass validation.** Otherwise, you'll get a birth date mismatch error ([`OCE012`](./errors#oce012) error code).
