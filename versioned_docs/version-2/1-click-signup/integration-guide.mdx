## Setup

<CollapsableGroup>
<CollapsableSection>
<CollapsableHeader>
### 1. Access [Dashboard](https://dashboard.verified.inc)
</CollapsableHeader>

Follow these steps to access the Verified Dashboard:
1. Login using your email.
2. Enter the verification code sent to your email to gain access.

</CollapsableSection>

<CollapsableSection>
<CollapsableHeader>
### 2. Configure Brand Settings
</CollapsableHeader>

On the Brand Details page, configure:
- **Integration Type**: Determines how 1-Click Signup is integrated.
- **Default Credential Requests**: Adjust the type of data you want Verified to source for your brand.

</CollapsableSection>
</CollapsableGroup>

## Integration

<CollapsableGroup>
<CollapsableSection>
<CollapsableHeader>
### 1. Prompt User for Phone Number
</CollapsableHeader>

Create a UI form for collecting phone numbers. Ensure proper validation and formatting.

</CollapsableSection>

<CollapsableSection>
<CollapsableHeader>
### 2. Call `POST /1-click`
</CollapsableHeader>

Send a request to the Verified API with the user’s phone number. The response includes a URL and verification code.

</CollapsableSection>

<CollapsableSection>
<CollapsableHeader>
### 3. Send SMS to User
</CollapsableHeader>

Use the verification code from the API response to send an SMS to the user.

</CollapsableSection>

<CollapsableSection>
<CollapsableHeader>
### 4. Redirect User
</CollapsableHeader>

Redirect users to the Verified-hosted page using the URL from the API response.

</CollapsableSection>
</CollapsableGroup>

## Go Live!

<CollapsableGroup>
<CollapsableSection>
<CollapsableHeader>
### Final Steps to Go Live
</CollapsableHeader>

1. Set your brand’s environment to **Production** in the Verified Dashboard.
2. Submit a request for API approval.
3. Replace the Sandbox API key with the Production API key in your application.

</CollapsableSection>
</CollapsableGroup>
