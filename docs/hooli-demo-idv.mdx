The open source Hooli demo demonstrates an example implementation of the [issuing](https://www.unumid.co/issue) Unum ID digital identity cards which unlocks [Free IDV](/free-idv-guide) offering. 

It is an application for a fictional customer, Hooli, which uses Unum ID to issue reusable <Tip type="credential">credentials</Tip> to its users. Additional information on how to issue credentials can be found in the API's issue credentials [documentation](/api-overview#issue-credentials).

## Web Demo
The live Hooli web application demo can be found [here](https://hooli-web.demo.sandbox-unumid.co/login). Its open source code can be found [here](https://github.com/UnumID/Hooli-Demo-Web).

## Mobile Demos
The mobile demo showcase how the Unum ID Web Wallet interacts within a native app to achieve the same functionality as the web demo.

### iOS
The Hooli iOS Demo source code can be found [here](https://github.com/UnumID/Hooli-iOS).

### Android
The Hooli Android Demo source code can be found [here](https://github.com/UnumID/UnumIDWebWallet-android).
