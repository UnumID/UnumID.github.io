The open source Hooli demo demonstrates an example implementation of the [issuing](https://www.verified.inc/issue) Verified Inc digital identity cards which unlocks the [Free IDV](/issuance-guide) offering. It is an application for a fictional customer, Hooli, which uses Verified Inc to issue reusable <Tip type="credential">credentials</Tip> to its users. 

Additional information on how to issue credentials can be found in Free IDV [guide](/issuance-guide) and in the API's issue credentials [documentation](/api-overview#issue-credentials).

## Web Demo
The live Hooli web application demo can be found [here](https://hooli-web.demo.sandbox-verifiedinc.com/login). Its open source code can be found on [Github](https://github.com/UnumID/Hooli-Demo-Web).

## Mobile Demos
The mobile demo showcase how to achieve the same functionality as the web demo in a native app.

### Android
The Hooli Android Demo can be installed from [here](https://appdistribution.firebase.dev/i/f59869e16365c7f8). The source code can be found on [Github](https://github.com/UnumID/UnumIDWebWallet-android).

### iOS
The Hooli iOS Demo can be installed from [here](https://testflight.apple.com/join/kjdN0EAD). The Hooli iOS Demo source code can be found on [Github](https://github.com/UnumID/Hooli-iOS).
