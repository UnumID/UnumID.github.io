The open source Hooli demo demonstrates an example implementation of the [issuing](https://www.unumid.co/issue) Unum ID digital identity cards which unlocks the [Free IDV](/free-idv-guide) offering. It is an application for a fictional customer, Hooli, which uses Unum ID to issue reusable <Tip type="credential">credentials</Tip> to its users. 

Additional information on how to issue credentials can be found in Free IDV [guide](/free-idv-guide) and in the API's issue credentials [documentation](/api-overview#issue-credentials).

## Web Demo
The live Hooli web application demo can be found [here](https://hooli-web.demo.sandbox-unumid.co/login). Its open source code can be found on [Github](https://github.com/UnumID/Hooli-Demo-Web).

## Mobile Demos
The mobile demo showcase how to achieve the same functionality as the web demo in a native app.

### iOS
The Hooli iOS Demo source code can be found on [Github](https://github.com/UnumID/Hooli-iOS).

### Android
The Hooli Android Demo can be installed from Firebase [here](https://appdistribution.firebase.dev/i/f59869e16365c7f8). The source code can be found on [Github](https://github.com/UnumID/UnumIDWebWallet-android).
